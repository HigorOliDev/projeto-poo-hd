Sobre o Projeto

Este site foi desenvolvido como parte do projeto da disciplina de Programação Orientada a Objetos (POO). O objetivo do site é fornecer uma solução eficiente para o gerenciamento escolar, abrangendo recursos como o cadastro de turmas, alunos, professores e o controle de informações escolares.

Autores

  • David Silva
  • Higor

Objetivo

Criar um sistema que facilite a administração de uma escola, oferecendo funcionalidades como:

  • Gerenciamento de turmas e alunos
  • Cadastro de professores e disciplinas
  • Organização de horários e salas
  • Consulta e edição de informações de forma rápida e segura

Tecnologias Utilizadas

O desenvolvimento do site contou com o uso das seguintes tecnologias:

  • HTML5, Bootstrap e CSS3 para a estrutura e estilo
  • Python com Django para o backend
  • Banco de dados SQLite para armazenamento de informações
